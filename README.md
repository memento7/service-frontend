# service frontend
memento web frontend source

using [jikji 2](https://github.com/Prev/jikji)



## How to run
Install jikji

```
$ pip3 install jikji

or

$ git clone https://github.com/Prev/jikji.git
$ pip3 install ./jikji
```

Run application
```
$ jikji . generate
or
$ jikji . listen
```


(*) To generate memento service, you need to configure authorization key on `lib/security.py`
